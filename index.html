🔧 Installation Instructions

  1. Upload the entire cfa-fire-forecast-plugin folder to your WordPress /wp-content/plugins/ directory
  2. Activate the plugin through the WordPress admin panel under Plugins
  3. Configure settings in Settings > CFA Fire Forecast
  4. Add the shortcode to any post or page where you want the forecast to appear

📝 How to Use

Basic Usage:

[cfa_fire_forecast]

With Options:

[cfa_fire_forecast district="north-central-fire-district" show_scale="true" auto_refresh="true"]

Available Options:

  • district - Fire district to display (e.g., "north-central-fire-district")
  • show_scale - Show fire danger ratings scale (true/false)
  • auto_refresh - Enable refresh button (true/false)

🏗️ Technical Features

  • WordPress HTTP API - Secure data fetching using WordPress native functions
  • Transient Caching - Efficient data storage using WordPress built-in caching
  • WordPress Cron - Automated scheduling using WordPress native cron system
  • PHP DOMDocument - Reliable HTML parsing for data extraction
  • AJAX Refresh - Real-time data updates without page reload
